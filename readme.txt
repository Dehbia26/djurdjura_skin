1. DESCRIPTION DU SITE :
Djurdjura Skin est un mini site e-commerce proposant des produits cosmétiques naturels à base d’ingrédients de Kabylie (huiles, savons, masques, coffrets). 
Le site met en avant les traditions ancestrales des montagnes du Djurdjura à travers une interface moderne, responsive et 100% côté client (HTML, CSS, JavaScript).

2. PAGES DU SITE (conformes aux exigences) 

- index.html         : page d’accueil
- inscription.html   : formulaire d’inscription (validation RegEx)
- connexion.html     : formulaire de connexion (localStorage)
- produits.html      : affichage dynamique des produits (JSON + filtres)
- commande.html      : panier + formulaire de livraison

Toutes les pages partagent un menu de navigation commun.


3. FONCTIONNALITÉS IMPLÉMENTÉES

✔ Affichage dynamique des produits depuis un fichier JSON
✔ Filtrage côté client par catégorie (huiles, savons, masques, coffrets)
✔ Système de panier avec localStorage
✔ Formulaire de commande avec validation des champs (RegEx)
✔ Inscription et connexion utilisateur (RegEx + localStorage)
✔ Session utilisateur simulée
✔ Design responsive (Flexbox, Grid, media queries)
✔ Menu de navigation identique sur toutes les pages
✔ Code structuré (fichiers HTML/CSS/JS séparés)
✔ Utilisation des balises sémantiques HTML5 (<header>, <nav>, <main>, <section>, <footer>)


4. INSTRUCTIONS D’UTILISATION

1. Lancer le site avec Live Server (clic droit sur index.html → Open with Live Server)
2. Créer un compte via la page "Inscription"
3. Se connecter avec les identifiants créés
4. Parcourir les produits sur la page "Nos produits"
5. Filtrer par catégorie (Huiles, Savons, Masques, Coffrets)
6. Ajouter des produits au panier
7. Visualiser le panier sur la page "Commande"
8. Remplir le formulaire de livraison et valider la commande
